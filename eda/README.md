# Exploratory Data Analysis (EDA)

Analysis of the GTSinger (vocal) and RAVDESS (emotion) datasets used in this project.

## Files

- `eda.py` — Main EDA script. Generates summary statistics and plots for both datasets.
- `eda_plots/` — Output plots generated by `eda.py`.
- `GTSinger_sample_50/` — 50 sample audio files from the GTSinger dataset.
- `requirements.txt` — Python dependencies for this module.

## Usage

```bash
pip install -r requirements.txt
python eda.py
```

## Prerequisites

- Place the GTSinger sample data in `GTSinger_sample_50/`
- Place the RAVDESS dataset in an `archive/` folder (same level as `eda.py`)
